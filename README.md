# 🛡️ OWASP Juice Shop – Vulnerability Assessment & Penetration Testing (VAPT)

## 📌 Project Overview

This repository contains a professional **Web Application Vulnerability Assessment & Penetration Testing (VAPT)** report conducted against the **OWASP Juice Shop** application in a controlled laboratory environment.

The assessment demonstrates practical web application security testing using industry-standard methodologies and tools, with findings mapped to **OWASP Top 10:2025**, **CVSS v3.1**, and **MITRE CWE**.

# 🔍 Key Vulnerabilities Identified

- SQL Injection
- Authentication Bypass
- Broken Access Control
- Information Disclosure
- Hidden Route Discovery
- DOM-Based Cross-Site Scripting (DOM XSS)
- Reflected Cross-Site Scripting (XSS)
- Sensitive File Exposure
- Weak Password Hashing (MD5)
- Sensitive Information Disclosure through JWT
- Exposed Application Configuration API
- Chained Account Takeover
